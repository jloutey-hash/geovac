# Combined Correction: Attenuation + Fock-Weighted Kinetic

**Date:** 2026-03-12

## Method

```
V_cross(a) *= 1 / (1 + alpha_eff(a) * S_a)    [attenuation]
  alpha_eff(a) = a0 * (n/Z)^2                  [Fock-style]
h1_diag(a) += lambda * S2w(a)                  [kinetic correction]
  S2w = sum_b S(a,b)^2 * (Z_A/n_a)^2 * (Z_B/n_b)^2  [Fock-weighted]
```
