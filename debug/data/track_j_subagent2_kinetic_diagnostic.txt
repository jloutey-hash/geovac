Track J Sub-agent 2: Associated Laguerre Algebraic Radial Solver
======================================================================
Date: 2026-03-29

Summary: Derived and implemented algebraic kinetic matrix elements for the
associated Laguerre basis phi_n(xi) = (xi-1)^{|m|/2} e^{-alpha(xi-1)} L_n^{|m|}(x),
completing the algebraic Level 2 radial solver for m!=0 (pi, delta, ... states).

Key derivation: Using x dL_n^(a)/dx = n L_n^(a) - (n+a) L_{n-1}^(a), the
derivative decomposes as g_n = sum_k (G0[n,k] + G1[n,k] x) L_k^(a)(x) with
bidiagonal G0 and diagonal G1. The kinetic integral then reduces to:
  K = -beta^{-(a+1)} [G0 P0 G0^T + G0 P1 G1^T + G1 P1 G0^T + G1 P2 G1^T]
where P_k = M_k + 2*beta * M_{k-1} are weighted moment matrices.

The centrifugal term uses the "lowered Stieltjes" identity:
  int x^{a-1}/(x+c) e^{-x} L_i L_j dx = (M_{-1} - J(a,c)) / c

All matrices from Sub-agent 1 infrastructure: M0, M1, M2 (associated moments),
M_{-1} (lowered moment), J (Stieltjes integral).
