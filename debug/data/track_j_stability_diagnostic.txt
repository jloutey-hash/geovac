Track J — Associated Laguerre Moment Machinery: Stability Diagnostic
=====================================================================
Date: 2026-03-29

Phase 1: Associated Laguerre Moment Matrices M0, M1, M2
  - Algebraic (exact from three-term recurrence)
  - Accuracy vs quadrature: < 1e-13 for all alpha=0-3, N=5-30
  - M0: diagonal (orthogonality)
  - M1: tridiagonal (single recurrence)
  - M2: pentadiagonal (double recurrence)

Phase 2: Lowered Moment Matrix M_{-1}
  - Algebraic (exact from DLMF 18.9.13 identity)
  - L_n^(alpha) = sum_{k=0}^n L_k^(alpha-1), so M_{-1}[i,j] = sum_{p=0}^{min(i,j)} h_p^(alpha-1)
  - NOT tridiagonal (dense lower-triangular-constant structure)
  - Accuracy vs quadrature: < 1e-13 for all alpha=1-3, N=5-30

Phase 3: Stieltjes Integral Matrix J
  - Hybrid: Miller backward recurrence for S_n + forward/backward for J rows
  - Base case: e^a E_1(a) with asymptotic series for a > 500

  Key observations:
  - For a >= 4: machine precision (< 1e-13) at ALL N, independent of N
    The backward recurrence in the right segment dominates and is perfectly stable.
  - For a = 2: < 1e-10 up to N=25 (adequate for production N <= 20)
  - For a = 1: degrades like O(N^1.5) due to forward recurrence in left segment
    At N=20: 1.3e-8 (marginal); at N=30: 8.6e-8 (still usable)
  - Root cause of a < 2 degradation: forward recurrence from j=0 to j=i
    amplifies rounding errors. The amplification factor depends on (2j+alpha+1+a)
    vs (j+alpha) — smaller a gives less dominant diagonal, worse amplification.

  Physical use case: centrifugal decomposition gives a = 4*alpha_basis
  where alpha_basis = sqrt(c^2) ~ 0.5-2.0, so a ~ 2-8.
  At these parameters: < 1e-10 accuracy guaranteed for N <= 30.

Algorithm summary:
  1. S_0^(0)(a) = e^a E_1(a) [or asymptotic series for a > 500]
  2. S_0^(alpha)(a) via upward recurrence: S_0^(k) = Gamma(k) - a*S_0^(k-1)
  3. S_1 = (alpha+1+a)*S_0 - Gamma(alpha+1)
  4. S_n for n >= 1 via Miller backward recurrence (stable for minimal solution)
  5. J[i, j<=i]: forward recurrence from j=0 (using S_i as seed)
  6. J[i, j>i]: backward recurrence from j=M (homogeneous, normalized using J[i,i+1])
  7. Symmetrize: J = (J + J^T) / 2
