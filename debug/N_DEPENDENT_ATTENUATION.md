# N-Dependent Cross-Nuclear Attenuation

**Date:** 2026-03-12

## Motivation

Uniform pade(0.5) gives nearly exact H2 force constant (6.9% err) but
overcorrects R_eq. For LiH, it creates a valid minimum at R=4.68 (expt 3.02).
Core orbitals should be attenuated less than valence.

## Method

```
alpha_eff(a) = alpha0 * (n_a / Z_a)^beta
V_cross(a) *= 1 / (1 + alpha_eff(a) * S_a)
```

- Core (Li 1s, n/Z=0.33): alpha_eff small -> minimal attenuation
- Valence (H 1s, n/Z=1.0): alpha_eff = alpha0 -> full attenuation
