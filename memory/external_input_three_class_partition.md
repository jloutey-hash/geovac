The multi-focal-composition wall pattern has 6 instances (5 physics: H1, LS-8a, HF-3, HF-4, HF-5; 1 chemistry: W1e). All six share the unifying reason: the framework determines the structural skeleton but does not autonomously generate calibration data. But *outside* that boundary, the external-input space partitions into **three categorically distinct classes** with different closure paths:

**Class 1 — Calibration parameter values** (UV-physics-tied or AC-inner-factor-tied)
- Members: H1 (Yukawa values), LS-8a ($Z_2$, $\delta m$ counterterms), HF-5 (multi-loop QED)
- Closure path: requires external physics from outside the framework (UV completion, Standard-Model phenomenology)
- Structurally bounded — no autonomous extension of the framework's primitive operations can generate these
- "Second packing axiom" is the speculative path forward

**Class 2 — Multi-focal composition** (spatial × spatial composition rules)
- Members: HF-3 (recoil cross-register $V_{ne}(\hat r_e, \hat R_n)$), HF-4 (Zemach magnetization-density, operator-level largely closed by W1b extension)
- Closure path: architectural extension within the framework's existing computational primitives (multi-week to multi-month)
- Framework-internal-extendable — the closure path is well-defined; the bare axiom set just doesn't autonomously execute it

**Class 3 — Multi-determinant correlation** (N-representable density-matrix manifold beyond zero-entropy class)
- Members: W1e (NaH inner-region overattraction after F1-F6 + Schmidt + core correlation)
- Closure path: chemistry-methods on the N-representable density-matrix manifold (explicit-core FCI, CCSD, DMRG)
- **Ambiguous between framework-internal-extendable (Reading A) and architecturally-scoped (Reading B)** — the load-bearing diagnostic is the explicit-core 12-electron HF on NaH (~2-4 weeks)

**Why:** The diagnosis came from the chemistry-positioning sprint (2026-05-25) where two diagnostic tracks ran in parallel. Track 1 (W1e structural reading) split walls into calibration-external vs framework-internal-extendable; Track 2 (modular → chemistry, clean negative) sharpened the split into three classes by noting W1e's multi-determinant residual is categorically distinct from both H1's UV-tied calibration AND HF-3's spatial composition. The three classes have different mathematical character: Class 1 lives outside the framework's scope structurally; Class 2 is architectural-extension work the framework's primitives could execute; Class 3 is correlation-physics on a density-matrix submanifold the framework's spectral-triple machinery does not natively engage.

**How to apply:** When a new sprint surfaces a wall, ask which of the three classes it belongs to — the closure path depends on the class. Class 1 walls take "no autonomous closure" verdicts; Class 2 walls take architectural-extension-sprint plans; Class 3 walls take chemistry-methods-or-equivalent. Pattern: don't propose Class-2 closure mechanisms (e.g., PK, magnetization extension) for Class-3 walls — they project onto the zero-entropy density-matrix submanifold and saturate below the wall.

The chemistry analog of "second packing axiom" (Class 1 speculative path) is "different architecture beyond composed two-electron active space" — this is the Reading B verdict if the explicit-core HF diagnostic confirms it. Reading A would put W1e in Class 2 (framework-internal-extendable) rather than its own distinct class.
