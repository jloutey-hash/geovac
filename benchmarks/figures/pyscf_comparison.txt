==============================================================================
SUMMARY: Where GeoVac Wins and Loses
==============================================================================

STRENGTHS:
  - Speed: GeoVac is extremely fast for single-electron systems
    (sparse eigenvalue problem, no integral evaluation)
  - Scaling: O(V) with >97% sparsity vs O(N^4) for HF integrals
  - Simplicity: No basis set optimization, no integral libraries
  - Single-electron accuracy: Converges to exact H, He+, Li2+ etc.
  - Theoretical foundation: 18/18 symbolic proofs that graph
    Laplacian is conformally equivalent to S3 Laplace-Beltrami

WEAKNESSES / CURRENT LIMITATIONS:
  - H2 Full CI accuracy ~2%: Mulliken cross-nuclear is an
    approximation. Exact orbital integrals would reduce error.
  - Multi-electron: Mean-field He uses Z-scaled kinetic_scale;
    no general DFT/HF multi-electron solver yet.
  - No basis-set extrapolation procedure established.
  - V_ee model (point-charge coordinates) needs improvement
    for higher accuracy; current result benefits from error
    cancellation between V_cross and V_ee approximations.

PREVIOUSLY DIAGNOSED AND FIXED (Feb 2026):
  1. Bridge architecture: _get_boundary_states_prioritized()
     now returns ALL states sorted by (n,l,|m|), connecting
     n=1 core states first. Bonding/antibonding splitting
     restored: 0.088 Ha splitting at max_n=5.
  2. Cross-nuclear V_en: Replaced point-charge model with
     Mulliken approximation + <1/r> cap for large-n states.
     V_cross(n=1) = -0.538 Ha (vs exact -0.61 Ha, 88%).
  3. Double-counting: Full CI now uses pure-kinetic H1 = kL,
     no node weights W. Old code double-counted nuclear
     attraction, producing -1.97 Ha (68% error).
  4. Variational collapse: Added <1/r> cap on V_cross so
     diffuse high-n states do not accumulate full -Z/R.

HONEST VERDICT:
  GeoVac's core claim — O(V) sparse graph eigenvalue replacing
  O(N^4) integral evaluation — is validated. Single-electron
  systems converge to <0.1% with no basis-set fitting. H2 Full
  CI now achieves ~2% error with monotonic basis convergence,
  comparable to PySCF FCI/STO-3G (~3.2%) on accuracy while
  using a fundamentally different (topology-based) approach.
