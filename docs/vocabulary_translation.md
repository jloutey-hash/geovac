# GeoVac Vocabulary Translation Table

**Purpose:** Maps GeoVac-internal vocabulary to standard mathematical/physical terminology.
Use when writing outreach documents (N1/N2/N3 notes), scrubbing paper introductions, or
responding to external readers. Three columns: Internal term | Standard rendering | Gloss/caveat.

---

| Internal term | Standard rendering | Gloss / caveat |
|:---|:---|:---|
| Fock graph / GeoVac graph | Truncated Camporesi–Higuchi spectral triple on SU(2); concretely, the Peter–Weyl basis on S³ truncated at shell n ≤ n_max with shell-adjacency structure | The adjacency is graph-Laplacian; the spectrum reproduces the Coulomb/hydrogen quantum numbers via Fock's 1935 SO(4) symmetry |
| Focal length (of a projection) | Scale/projection parameter λ of the stereographic (Fock) projection S³ → ℝ³; fixes the energy shell E = −λ²/2 | Each observable corresponds to a choice of λ; "multi-focal" means multiple λ values used simultaneously |
| Layer 1 | Combinatorial spectral data of the bare graph: eigenvalues, degeneracies, Gaunt selection rules — all in ℤ or ℚ, no continuum | Synonymous with "discrete substrate" or "skeleton"; no physics input required |
| Layer 2 | Physical observable obtained by projecting Layer 1 data via a named map (stereographic, Hopf, heat-kernel, etc.) | Where transcendental constants enter; each projection is catalogued in Paper 34 |
| Master Mellin engine, M1/M2/M3 | Heat-kernel Mellin transforms Tr(D^k · e^{−tD²}) at operator order k = 0, 1, 2, classified as: M1 (Hopf-base measure, Vol(S²)/π² = 4/π²), M2 (Seeley–DeWitt coefficients, ring √π·ℚ ⊕ π²·ℚ), M3 (eta-type vertex-parity Hurwitz, Catalan G and β(2k)) | Framework-internal taxonomy in Paper 18 §III.7; standard name for the underlying objects is "heat-kernel expansion of a Dirac-type operator" |
| Exchange constant | Transcendental number (π, ζ(3), G, …) that enters a specific continuum projection irreducibly; classified by which Layer-2 map generates it | Weyl–Selberg language; analogous to "calibration parameter" but with a precise spectral-geometric source identified |
| Projection (Paper 34 sense) | Named evaluation map from discrete spectral data to a physical observable (e.g., stereographic, Hopf-bundle push-forward, heat-kernel trace, temporal compactification) | Not a quantum-mechanical projector; 28 projections catalogued in Paper 34 with three-axis tagging |
| Structural skeleton | Spectral and representation-theoretic structure forced by the geometry alone: eigenvalue degeneracies, angular selection rules (Gaunt), Hopf bundle, Dirac spinor sector, SU(2)/SU(3) gauge structure | Corresponds roughly to "model-independent / basis-intrinsic" structure in chemistry/QI language |
| Calibration data | External numerical input not derivable from the geometry: Yukawa couplings, nuclear magnetic moments, empirical mass ratios | Paper 18 §IV.6 "inner-factor input data" tier; synonymous with "measurement input" |
| Natural geometry | The coordinate system in which the Schrödinger equation (or its relativistic analog) separates for a given electron configuration | e.g., S³ for one-electron atoms, prolate spheroidal for H₂⁺, hyperspherical for two-electron atoms |
| Composed geometry | A fiber-bundle product of per-electron-group separable coordinate systems; each group gets its own natural geometry, coupled via Z_eff screening and a pseudopotential | "Level 5" in the natural-geometry hierarchy; roughly analogous to LCAO with per-fragment basis sets, but each fragment basis is the exact natural-geometry basis for that fragment |
| PK (pseudopotential) | Phillips–Kleinman pseudopotential; projects out occupied core orbitals from the valence Hamiltonian | Standard quantum chemistry concept; GeoVac uses an ab initio variant derived from the core eigenvectors |
| Bit-exact | Result obtained in exact rational/integer arithmetic, or with IEEE 754 float64 residual = 0.0 exactly | Stronger than "machine precision"; implies no floating-point cancellation; reserved for Layer 1 skeleton results |
| π-free | Spectral data lying in ℤ, ℚ, or a real quadratic extension ℚ(√d) — no transcendental numbers | A property of Layer 1 (the bare graph); lost at Layer 2 when projections are applied |
| Graph-native | Computed entirely on the discrete substrate (Layer 1), without any continuum projection | Antonym: "projected" or "layer-2" |
| Propinquity (Papers 38/45) | Van Suijlekom's state-space Gromov–Hausdorff distance Λ on quantum metric spaces | NOT Latrémolière's propinquity (Papers 38 v2 / 45 v2 carry explicit errata on this attribution); the framework uses vS's Wasserstein–Kantorovich formulation |
| Achievable envelope | The subspace of B(H) reachable by finite products of truncated multiplier operators P_n f P_n within the operator system at cutoff n_max | Contrasted with "full envelope" = all of B(H); prop = 2 holds on the achievable envelope |
| Offdiag CH Dirac | The engineered off-diagonal modification D^{offdiag} of the Camporesi–Higuchi Dirac operator, constructed so that the Lipschitz kernel condition ker([D, ·]) ∩ Op = ℂ·1 holds | The standard (truthful) CH Dirac has a kernel too large for the Connes distance to be well-defined on cross-shell pairs; the offdiag modification fixes this |
| K⁺ weak form | Compression of the Krein inner-product structure to the positive eigenspace K⁺ of the Krein fundamental symmetry J_L, where the indefinite Krein form reduces to a positive-definite Hilbert inner product | The K⁺ seminorm is identically zero (Paper 45 annihilation theorem, 2026-06-09); "weak form" now labels the degeneracy theorem, not a propinquity bound |
| WH register | CLAUDE.md §1.7 internal working-hypotheses ledger; bold-claim register for reasoning between sessions | NOT paper-level claims; entries in the WH register have not been submitted to external review |
| Forced/free seam | The boundary between representation-theoretically forced spectral structure (structural skeleton) and external numerical input (calibration data) | Paper 57; analogous to the model-independent / model-dependent split in effective-field-theory language |
| Camporesi–Higuchi Dirac | The canonical Dirac operator on the round S³ = SU(2) spinor bundle, with spectrum |λ_n| = n + 3/2 and degeneracy g_n = 2n(n+1) | Standard construction; reference: Camporesi–Higuchi 1996, J. Geom. Phys. 20:1 |
| Propagation number (prop) | The Connes–van Suijlekom propagation number of an operator system: the smallest k such that Op^k = B(H) | prop = 2 for the GeoVac operator system on the achievable envelope; prop = ∞ on the full B(H) envelope |
| Peter–Weyl basis | The orthonormal basis of L²(G) for a compact group G given by matrix coefficients of irreducible unitary representations | On SU(2) = S³, this coincides with the (n, l, m_l) shell-labeling used throughout the framework |
| Fock projection / stereographic projection | The SO(4)-equivariant diffeomorphism (Fock 1935) that identifies the momentum-space hydrogen wavefunctions with functions on S³, turning the Coulomb problem into a spectral problem on the round sphere | Establishes the conformal equivalence; the "focal length" is the stereographic scale parameter p₀ = √(−2E) |
| Seeley–DeWitt coefficients | The heat-kernel expansion coefficients a_k appearing in Tr(e^{−tD²}) ~ Σ a_k t^{(k−d)/2} as t → 0⁺ | Standard spectral geometry; M2 sub-mechanism in the master Mellin engine |
| Multi-focal-composition wall | A class of observables where the framework produces correct angular structure but cannot autonomously determine the radial coupling values | Documented negative result in 6 independent settings (Yukawa, two-loop SE, hyperfine, W1e chemistry); analogous to "calibration-data boundary" |
| Truncated operator system | The space Op_{n_max} = P_{n_max} C^∞(S³) P_{n_max} ⊂ B(H_{n_max}), an operator system in the sense of Connes–van Suijlekom (not a C*-algebra) | Finite-cutoff discretization of the algebra of observables; the correct mathematical object for finite-n_max truncations |
| Fock-graph shell / n-shell | The set of nodes with principal quantum number n; the eigenspace of the graph Laplacian with eigenvalue −(n²−1) | Standard hydrogen-atom notation; "shell" = "n-shell" = one full principal-quantum-number level |
| Hopf bundle structure | The principal U(1)-bundle π: S³ → S² given by the Hopf fibration; appears as the gauge structure of the GeoVac graph | Standard differential geometry; the Hopf bundle forces U(1) gauge structure on the graph |
